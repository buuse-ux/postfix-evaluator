# postfix-evaluator

A Java program that evaluates postfix (RPN) expressions.

## postfix dönüşümünü yani verilen operandlar ve operatörlerden oluşan bir ifadenin postfix kuralına göre dönüşümünü saglayan bir program.Postfix bize ifadenin operatörünün en sonda en az öncelikli olması gerektiğini söyler. Bu sistem agaclar tree konusunda sema seklinde de ifade edilirken kullanılabilir oncelikler sol sag kök seklinde ilerlenerek verilen  sıra numaralarının  izdüşümleri bize postfix ifadeyi verir.
